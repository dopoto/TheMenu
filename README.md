# TheMenu

A sample .NET 6 / Angular project.


## Setting up a development environment

### Prerequisites

- Visual Studio 2022
- SQL Server 2019+
- Angular dev environment
- A Google OAuth app (https://developers.google.com/identity/sign-in/web/sign-in)
